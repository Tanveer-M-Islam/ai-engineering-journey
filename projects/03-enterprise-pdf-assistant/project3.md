# Enterprise PDF Knowledge Assistant

An AI Engineering mini project using:

- RAG
- FAISS
- Embeddings
- Semantic Search
- PDF Ingestion
- Ollama LLMs

## Features

- PDF document ingestion
- Chunking
- Semantic retrieval
- Vector database storage
- Retrieval-Augmented Generation (RAG)

## Tech Stack

- Python
- Sentence Transformers
- FAISS
- Ollama
- PyMuPDF

## How to Run

### Build Vector Index

python build_index.py

### Start Chat

python search.py
