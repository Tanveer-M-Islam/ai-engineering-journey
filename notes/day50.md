# Day 50 – LangGraph Introduction

## What is LangGraph?

LangGraph is a framework for building stateful AI applications.

Instead of writing many if/else statements, you connect nodes into a graph.

## Core Concepts

- State
- Node
- Edge
- START
- END

## Execution Flow

START

↓

Node

↓

END

## Why LangGraph?

- Cleaner architecture
- Stateful workflows
- Easy to extend
- Better for multi-step AI systems
