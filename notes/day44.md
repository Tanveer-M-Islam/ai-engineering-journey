## Multi-Step Agents

A multi-step agent can use more than one tool before generating a final answer.

Workflow:

Question
↓
Tool Call
↓
Tool Result
↓
Tool Call
↓
Tool Result
↓
Final Answer

Benefits:

- Handles complex tasks
- Supports multiple tools
- Closer to real AI agents

Examples:

get_time()
get_date()
get_tomorrow_date()
multiply()
